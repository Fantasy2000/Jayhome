# JayHome 个人网站项目

这是一个简单易用的个人主页网站，基于https://github.com/ZYYO666/homepage进行二开的，新增了包含评论系统、轮播图、音乐播放器、弹幕滚动、后台管理等功能。

## 🎯 增加的功能

- **简洁美观的首页**：展示个人信息和作品
- **实时评论系统**：支持访客留言和互动
- **后台管理系统**：轻松管理网站内容、评论和用户
- **弹幕功能**：支持实时弹幕互动
- **留言板**：访客可以留下祝福和建议
- **Canva背景**：美观的背景设计
- **轮播Banner**：动态展示重要内容
- **音乐播放器**：支持背景音乐播放
- **响应式设计**：适配手机和电脑等不同设备

## 📦 如何使用

### 1. 安装环境

你需要准备：
- 一台服务器（推荐使用宝塔面板）
- MySQL数据库
- PHP 7.4+ 环境

### 2. 上传文件

将项目文件上传到你的服务器网站根目录。

### 3. 配置数据库

1. 在宝塔面板创建一个MySQL数据库
2. 导入 `database/comments.sql` 文件到数据库
3. 修改 `api/config.php` 文件中的数据库配置：
   ```php
   define('DB_USER', '你的数据库用户名');
   define('DB_PASS', '你的数据库密码');
   ```

### 4. 访问网站

- 首页：`http://你的域名/`
- 管理员后台：`http://你的域名/admin/`
  - 默认账号：admin
  - 默认密码：admin123

## 📝 简单说明

这是一个适合小白用户使用的个人网站模板，你可以根据自己的需求修改内容：

- 修改 `index.html` 中的文字和图片
- 在管理员后台添加轮播图和管理评论
- 替换 `static/img/` 目录下的图片
- 修改 `music/` 目录下的背景音乐
